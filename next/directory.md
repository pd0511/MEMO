# Directory

暫定ディレクトリ構成

```
root
├── public
└── src/
    ├── app
    ├── components/
    │   ├── atoms
    │   └── layouts
    ├── config
    ├── hooks
    ├── icons
    ├── lib
    ├── templates
    ├── store
    ├── tailwind
    ├── types
    └── util
```

- atoms - コンポーネント
	- atoms - パーツ
	- layouts - 組み合わせ
- config - 使い回す変数
- hooks - hooks用
- icons - Iconまとめ
- lib - ライブラリ設定
- templates - useEffect使うページを逃す
- store - 状態
- tailwind - tailwind用
- types - ts 型定義
- util - util
